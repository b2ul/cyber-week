# الإسبوع التوعوي للأمن الإلكتروني — نظام حفل الافتتاح

## الملفات
- `public/index.html` — شاشة المسرح (الشاشة الكبيرة)
- `public/unlock.html` — صفحة الجوال للجمهور
- `api/winner.js` — الخادم (يربط الجوالات بالشاشة)

---

## خطوات النشر — خطوة بخطوة

### الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى github.com
2. اضغط "Sign up"
3. أنشئ حساباً مجانياً

### الخطوة 2: رفع الملفات
1. بعد تسجيل الدخول، اضغط "New repository" (الزر الأخضر)
2. سمّه: `cyber-week`
3. اضغط "Create repository"
4. اضغط "uploading an existing file"
5. ارفع كل الملفات (مجلد public، مجلد api، vercel.json)

### الخطوة 3: النشر على Vercel
1. اذهب إلى vercel.com
2. اضغط "Sign up" وسجّل بحساب GitHub مباشرة
3. اضغط "Add New Project"
4. اختر مستودع `cyber-week`
5. اضغط "Deploy" — انتظر دقيقة
6. Vercel سيعطيك رابطاً مثل: `https://cyber-week.vercel.app`

---

## يوم الفعالية

### الشاشة الكبيرة (على الحاسوب المتصل بالبروجكتور):
- افتح المتصفح
- اذهب إلى: `https://cyber-week.vercel.app/index.html`
- اضغط F11 لملء الشاشة
- الصفحة ستعرض QR code تلقائياً

### الجمهور (على جوالاتهم):
- يمسحون QR code الظاهر على الشاشة
- يفتح عندهم صفحة الإدخال مباشرة

### تسلسل الحفل:
1. شغّل الفيديو على الحاسوب
2. أوقف الفيديو يدوياً عند اللحظة المناسبة
3. افتح صفحة index.html في tab جديد
4. الجمهور يمسح QR ويدخل أسماءهم والرمز (23)
5. أول إجابة صحيحة ← يظهر الاسم تلقائياً على الشاشة + عد تنازلي
6. شغّل الفيديو مجدداً لإكمال الحفل 🎉

## الرمز الصحيح
`23`


















ة 1 — GitHub (5 دقائق):

اذهبي لـ github.com ← سجّلي حساب جديد
اضغطي الزر الأخضر "New" ← سمّيه cyber-week ← "Create repository"
اضغطي "uploading an existing file" ← ارفعي الملفات الخمسة

الخطوة 2 — Vercel (3 دقائق):

اذهبي لـ vercel.com ← "Sign up with GitHub"
"Add New Project" ← اختاري cyber-week ← "Deploy"
تنتهي برابط مثل https://cyber-week-xxx.vercel.app 🎊


يوم الفعالية:

الشاشة الكبيرة: افتحي الرابط + F11
الجمهور: يمسحون QR اللي يظهر تلقائياً على الشاشة
أول واحد يكتب 23 ← اسمه يظهر على الشاشة فوراً 🔥
